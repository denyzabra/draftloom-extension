Enable Chrome AI 🤖

DraftLoom uses Chrome's built-in AI for 100% on-device processing. Quick 3-minute setup to unlock full functionality:

1 Open Chrome Flags

Paste into address bar and press Enter

2 Enable Required Flags

Search for and enable these flags:

  • Prompt API for Gemini Nano
  • Summarization API
  • Rewriter API
  • Translation API
3 Restart Chrome

Click "Relaunch" button in Chrome to apply changes

Note: After restart, Chrome will download the AI model (~2GB) automatically. This is a one-time download.
